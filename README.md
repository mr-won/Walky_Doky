# Walky_Doky
반려견 양육 &amp; 훈련 콘텐츠 앱 프로젝트
#### 서울 마포구
```
• 앱 개발 관련 경력 3년 이상 (또는 그에 준하는 실력)
• 2년 이상의 React Native 또는 하이브리드 앱 개발 프레임워크 경험 (Ionic, Flutter...)
• Git을 사용한 능숙한 형상관리 능력
• React 상태 관리 라이브러리 (Redux, MobX 등) 사용 경험 혹은 RxJS 사용 경험
```
```
• 결제 모듈을 포함한 서비스 개발 및 운영해 보신 분
• UI/UX에 관심이 많고 고민을 좋아하시는 분
• 채팅 또는 챗봇 프론트엔드 개발 경험이 있으신 분
• 스타트업 경험이 있으신 분
• 비 개발자 직군과 원활히 소통하는 능력
• 새로운 기술에 관심이 많고, 자기 개발을 위해 노력하시는 분
• 생산성 향상을 위한 개발 방법론 관한 실험적 사고
```
```

- 업무시간 30분을 줄이고 점심시간 30분을 늘렸어요.

- ️ 자율 출퇴근 제도

- 휴게공간 제공

- 스낵바 운영
- 실내 흡연실 운영 (냄새 완벽 차단! 비흡연자 걱정 No No !)
- 고사양 노트북 제공 (윈도우 or 맥북프로)
- 반려동물 동반출근 가능
- 허락이 필요 없는 자율적 연차 사용

- 도서구매비 지원
- ‍ 실력있는 동료들과의 코드 리뷰
```
